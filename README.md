# Kánon local app
Designed specifically for use with **kanon** web app.

# Start
`npm start`

# Build
`npm run build`

# Deploy
`npm run deploy`

# Options
 - `full`    - enables fullscreen mode
 - `dev`     - opens Dev tools ♥
 - `clearDB` - clear database (Warning! List of all used books will be lost!)
